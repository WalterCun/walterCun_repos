## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash

Generara variables de entorno para conexion con base de datos cucharachaDB
# En Windows
set PORT=3001
set DB_USER=[user]
set DB_PASSWORD=[pass]
set DB_HOST=[urldatabase] 
set DB_PORT=[port]
set DB_CLUSTER=[cluster]
set DB_NAME=[namedb]

# En Linux
$ export DB_USER=[user]
$ export DB_PASSWORD=[pass]
$ export DB_HOST=[urldatabase] 
$ export DB_PORT=[port]
$ export DB_CLUSTER=[cluster]
$ export DB_NAME=[namedb]

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
